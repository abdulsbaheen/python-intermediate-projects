# 🗂 File Organizer (Python)

A simple and powerful Python project that automatically organizes files into folders based on their type. This tool helps keep directories clean and structured by sorting files such as images, documents, videos, and more.

---

## ✨ Features

- 📂 Automatically organizes files  
- 🖼 Sorts images, documents, videos, music, and archives  
- 🧠 Detects file types using extensions  
- 📁 Creates folders automatically if not present  
- ⚡ Fast and efficient file handling  

---

## 🛠 Technologies Used

- Python  
- os module  
- shutil module  

---

## 🎯 Purpose

This project is part of my intermediate Python learning journey. It focuses on building real-world automation tools and improving my understanding of file handling and system-level operations.

---

## 🚀 How It Works

1. Scans all files in a selected folder  
2. Identifies file type based on extension  
3. Creates category folders if needed  
4. Moves files into appropriate folders
