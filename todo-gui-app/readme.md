# 📝 To-Do GUI App

A simple and interactive **To-Do List application** built using Python and Tkinter. This app helps users manage daily tasks with a clean graphical interface and supports saving tasks for future use.

---

## 📸 Screenshot

![App Screenshot]

## ✨ Features

- ➕ Add tasks  
- ❌ Delete selected tasks  
- 🧹 Clear all tasks  
- 🔄 Automatic task renumbering  
- 💾 Save tasks to file  
- 📂 Load tasks on startup  
- 🎨 Clean and simple GUI  
- 👨‍💻 Developer footer
